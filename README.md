🏠 California Real Estate Market Analysis
This project focuses on a comprehensive analysis of California property data using SQL Server for data engineering and Power BI for advanced visualization. The goal was to extract actionable insights regarding market trends, pricing dynamics, and data integrity.

🛠 Tech Stack
Database: MS SQL Server (CTEs, Window Functions, Data Cleaning)

Visualization: Power BI (DAX, Relationship Modeling, Conditional Formatting)

Data Source: California Sold Properties Dataset
